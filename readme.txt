== Description ==

PageStudio is a powerfull drag & drop page builder plugin for wordpress for creating responsive websites, no code, and no time wasted.

To know more about the plugin, visit the website [http://pagestudio.pro](http://pagestudio.pro).

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/checkcms` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
