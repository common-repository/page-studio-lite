# Version 1.0.6

  • new Added a new template system to your editor.
  • changed Added a default image for componentes with null or empty icons.

  • changed CSS Tweaked for more editor performance.

  • changed New option added to the welcome message to add some component or predefined element.

  • changed To avoid problem with the color value of all elements, the field now use a "color mask".
  • changed All Color system was remade and improved.
  • changed Now the gallery will display random images by default.
  • fix Fixed the right align bug with the grid in some sizes.

  • fix Fixed a problem of the image not aligning properly.

  • fix Fixed a rare problem when the editor not loading.
  • fix Fixed a rare problem when after editing the same object multiple times, the editor crashes.
  • fix Fixed a problem of the system not being able to remove the new row line when the page is clean.

  • fix Fixed a bug with the scrollbars going crazy inside the editor.

  • fix Fixed a bug when you clone some row and appears in the wrong place.
  • fix Fixed a bug where repeated colors are being rendered by the color history.
  • fix Fixed a bug where the content colors was not being correctly rendered by the editor, or rendering with missed colors.

  • fix Fixed a problem with the vimeo component not loading the movie.
  • fix Fixed the Notice: Undefined index: data-media, when trying to save the gallery with the default images.
